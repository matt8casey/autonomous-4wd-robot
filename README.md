# Autonomous 4WD Robot

A 3D-printed autonomous robotics platform built using an Elegoo Uno R3, ultrasonic sensing, Bluetooth control, and 4-wheel drive.

## Planned Features

- Manual Bluetooth control
- Obstacle detection
- Obstacle avoidance
- Custom CAD chassis
- Future sensor expansion
